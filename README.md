# Trabalho final web

## Instalação

### 1. Instalar todas as dependências
```bash
npm install
```

### 2. Executar o servidor em modo de desenvolvimento
```bash
npm run dev
```

O servidor estará disponível em `http://localhost:3000`

## Dependências Utilizadas

- Caso deseje instalar as dependencias individualmente.

#### better-sqlite3
Biblioteca SQLite para Node.js com melhor performance.
```bash
npm install better-sqlite3
```

#### cors
Middleware para habilitar CORS (Cross-Origin Resource Sharing).
```bash
npm install cors
```

#### express
Framework web minimalista para Node.js.
```bash
npm install express
```

#### morgan
Logger de requisições HTTP para Node.js.
```bash
npm install morgan
```

### Dependências de Desenvolvimento

#### nodemon
Ferramenta que reinicia automaticamente o servidor quando detecta mudanças nos arquivos.
```bash
npm install --save-dev nodemon
```
